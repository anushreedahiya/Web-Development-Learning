2. we can give multiple class name just separating the names through a space.
3. total height and width of an element will be the sum of margin+ border+ padding+ the element space itself.
4. Padding is imagined as within the element whereos margin is outside and hence no background-color will be applied there of the element.
5. Margin collapse concept states that when the margin of 2 element collide then the margin with highest value is given preference there and other margin is neglected. therefore margin only apply once.
