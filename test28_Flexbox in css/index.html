  1. Flex direction by default is left to right.
  2. There is no justify-items in flexbox.
  3. In flex-flow, we need to tell fex-direction first then flex-wrap value that can be wrap or wrap-reverse
  4. i.e., flex-flow: flex-direction flex-wrap
  5. More the order value, more the element is shifted to end.
  6. if in a group of 15 elements only 1 element order is defined then that element will go to end and rest remain unchanged as default order is 0.
  7. by default flex-grow is 0 and it means how much space will the element take in comparison to others.
  8. Flex-shrink is the ability of the element to shrink when the dimension of the page is changed and wrap isn't there.
  9. Align-self is used to align an element either to flex-start, end or at baseline. used when we want to override the configuration of a particular element
