importance example in css Specificity

Css follow cascade algorithm therefore it states or tells which property will be applied.

  1. Position or order of appreance: order in which rules appear
  2. Specificity: which css selectopr has strongest match, more specific one is given priority

    3. Inline selector (1000 value) > id selector (100 value) > classor attribute selector (10 value) > element selector (1 value) > universal selector (0 value)

    !important Specificity is 10,000 points therefore it is hard to overwrite it

  3. Origin: order in which css appears and where it come from, whether browser style
  4. Importance: Some css rule are weighted more heavily
